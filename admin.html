📍 Geocode Massal

Cari koordinat (lat/lng) otomatis untuk semua data yang belum punya, memakai link Maps kalau ada, atau cari lewat nama tempat via Photon (OpenStreetMap) kalau tidak. Ada jeda antar-request supaya tidak kena rate-limit.

📄 Hasil run terakhir (0 data)
💡 Data dengan link Maps yang sudah berisi koordinat (ada @lat,lng) akan langsung terisi tanpa perlu cari nama. Data dengan link short (maps.app.goo.gl) atau tanpa link sama sekali akan dicari lewat nama + kawasan, dicoba beberapa variasi query di Photon lalu fallback ke Nominatim (OpenStreetMap) kalau masih nihil — hasil di luar area Banyuwangi otomatis dilewati. Karena ada beberapa percobaan per data, prosesnya lebih lama dari sebelumnya tapi tingkat keberhasilannya lebih tinggi. Yang masih gagal setelah ini kemungkinan besar memang belum terdaftar di OpenStreetMap dan perlu input koordinat manual.
🔍 Audit Data Mencurigakan

Cek data yang SUDAH punya koordinat tapi nama/kawasannya menyebut kota lain (Jember, Situbondo, dll) — kemungkinan besar hasil geocode-nya salah nyantol ke tempat lain yang kebetulan ada di dalam radius Banyuwangi (bukan tempat aslinya). Ini beda dari statistik di atas yang cuma hitung yang belum ada koordinat.

📤 Import Koordinat Manual

Sudah isi manual kolom Lat/Lng di file hasil export tadi (Excel atau JSON)? Upload lagi di sini — sistem cocokkan lewat kolom ID, cuma baris yang lat/lng-nya terisi yang akan diproses.
